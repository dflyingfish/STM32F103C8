/******************************************************************************
 - Use the ST Motor Control Workbench application to configure this
   STM32 PMSM FOC library v4.0 opening the WB file: 
   SDK40x-STM3210E-EVAL-MB459-Shinano_Examples.stmc contained in WB folder
   It has been created for STM32F10x and MB459 with Shinano motor.
 - From IAR, select the right example from drop box menu, for instance STM3210E-EVAL_SPEED
 - For further information open the README file inside the example folder i.e., 
   \..\STM32_SDK_Examples\Speed monitor\readme.txt
 - Press F8 to batch-build the entire workspace (select a proper project configuration, 
   according to the Example in use);
 - Read the messages window and download the firmware;
 - Run the example
